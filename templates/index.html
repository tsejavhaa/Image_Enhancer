About Image Enhancer

SwinIR is a state-of-the-art image restoration model based on Swin Transformer, published at ICCV 2021. It achieves top performance across multiple restoration tasks.

Supported Tasks

Super Resolution
Upscale images 2×, 3×, 4×, or 8× using classical or real-world SR models.
Denoising
Remove Gaussian noise (σ=15/25/50) from colour or grayscale images.
JPEG Artifact Reduction
Restore compression artefacts at JPEG quality 10/20/30/40 for colour and gray images.

Training

Use train.py to fine-tune any model on your own paired dataset:

python train.py \
  --task classical_sr \
  --variant x4 \
  --epochs 100 \
  --batch_size 8

├── pretrained_models/  # Downloaded .pth files
├── uploads/            # User uploads
└── outputs/            # Enhanced images
